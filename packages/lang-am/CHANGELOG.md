# @fidel-tools/lang-am Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-06-13

### Added
- **Cleaned stopword list**: Expanded to 435+ academic stopwords, excluding context-dependent words like "በፊት", "ቤት", and "ፊት".
- **Expanded abbreviation list**: Built out 578+ standard contractions and exceptions.
- **Affix deduplication**: Deduped all stemmer prefixes and suffixes.
- **Protected words list**: Protected proper nouns like "ኢትዮጵያ", "አፍሪካ", and "አዲስ አበባ" from stemming.
- **NPM Config**: Configured files mapping for registry publishing.
